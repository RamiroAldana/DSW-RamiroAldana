Proposito del sitio

Con el propósito de realiza de una forma interactiva, de facil acceso y que permita plasmar los conceptos importantes, asi como etiquetas y demas referentes a HTML, se ha creado este espacio web, aqui iniciaras en el maravilloso mundo del desarrollo web, aprendiendo desde cero el lenguaje de etiquetado HTML.

Que Veremos en esta primera seccion


1. Introduccion a HTML:
Ya definimos que html es un lenguaje de etiquetado para modelar y estructurar información, pero como nacio, por que es tan popular, como comenzar a trabajar con el? Esto lo veremos en el primero tema de Introduccion al HTML


2. Etiquetas básicas de HTML:
Comenzaremos a conocer las etiquetas básicas de HTML y a jugar un poco con ellas, comenzaremos a escribir nuestras primeras lineas de codigo, para crear nuestra primera pagina web


3. Atributos de HTML: :
Una vez conozcamos las etiquetas básicas de HTML, veremos los atributos y valores de estas que nos permitirán comenzar a estructurar nuestra primera pagina web
